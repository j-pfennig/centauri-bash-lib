Topic - Content - Documentation content by module - centauri-bash-lib
General:
The library documentation is structured into modules, functions and topics. Each module is a bash script, and functions are always contained in modules. Topics have no executable code.
Remarks:
The main module _centauri_bash_lib (or it's proxy) must be sourced by the user script, see Initialization. Other library modules can be loaded dynamically.
