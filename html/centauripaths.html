Script - centauripaths - Resolve/symlink path shortcuts to real filesystem paths
Usage:
centauripaths option... [name | name/path | path]... centauripaths --deleted [--table] [name | name/path | path]... centauripaths --list [--table] [--verbose]
Resolve/symlink path shortcuts to real filesystem paths.
Options:
-d --dryrun show what would be done, make no changes -h --help show this text -n --no do not prompt for confirmation, assume 'no' -q --quiet report errors only -v --verbose be verbose (use twice for debug) -y --yes do not prompt for confirmation, assume 'yes'
-A --abstract make an abstract (network) path -D --deleted return trash folder(s) -E --error unavaillable paths are errors -F --fatal unavaillable paths are fatal -I --list list known shortcut names -K --symlink create symlink from cwd to folder -L --logical returned paths should be logical names -N --name link link name, can be repeated -P --physical returned paths should be physical -S --style mask modifies path output formatting -T --tabular generate tabular output -U --unique remove duplicated folders -W --work work set working folder
Arguments:
name shortcut name, to get a list run: centauripaths --list path any path (must contain a '/' or follow 'name') link link name, overrides short name if present mask bit mask, 1 := print - if missing, 2 := quoted work working folder, can be a short name
Remark:
The shortcut definitions are loaded from "centauri-bash-lib" which is searched at "~/.config" "~" "$CEN_ROOT/default" or "/etc/default".
Examples:
centauripaths --list --tabular --verbose # list all names/paths centauripaths temp # current user's TMPDIR centauripaths --logical automount/alpha4_work # get a logical path centauripaths --work=/tmp --symlink disk data # make symlinks ... centauripaths --work=/tmp disk --name=d1 data --name=d2
