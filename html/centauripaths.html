Script - centauripaths - Kurzbezeichnungen zu Pfaden im Dateisystem auflösen
Aufruf:
centauripaths option... [name | name/path | path]... centauripaths --deleted [--table] [name | name/path | path]... centauripaths --list [--table] [--verbose]
Kurzbezeichnungen zu Pfaden im Dateisystem auflösen.
Optionen:
-d --dryrun Zeigen was getan wird, keine Änderungen vornehmen -h --help diesen Text ausgeben -n --no Nicht nach Bestätigung fragen, 'nein' annehmen -q --quiet Nur Fehler melden -v --verbose Viel Ausgabe (zweimal zum Testen) -y --yes Nicht nach Bestätigung fragen, 'ja' annehmen
--embed name Benutze name anstelle des Script Namens --info Vollen Pfad und Version ausgeben --pager prog Programm für seitenweise Ausgabe, 'none' für keines --trace Trace actions() oder run() --wait Nachfrage vor Beenden
-A --abstract Erstelle abstrakten (Netzwerk) Pfad -D --deleted Finde Müll Verzeichnis(se) -E --error Fehlende Pfade sind Fehler -F --fatal Fehlende Pfade sind Fatal -I --list gültige Kurzbezeichnungen auflisten -K --symlink Verknüpfe zu aktuellem Verzeichnis -L --logical Zurückgegebene Pfade haben logische Namen -N --name link Verknüpfungsname, wiederholbar -P --physical Zurückgegebene Pfade haben physische Namen -S --style mask Verändert Ausgabeformatierung für Pfade -T --tabular Ausgabe als Tabelle -U --unique Doppelte Verzeichnisse entfernen -W --work work Setze Arbeitsverzeichnis
Argumente:
name Kurzbezeichnung, für eine Liste: centauripaths --list path Pfadangabe (muss ein '/' enthalten oder auf 'name' folgen) link Verknüpfungsname, überschreibt vorhandene Kurzbezeichnung mask Bitmaske, 1 := Ausgabe - wenn fehlend, 2 := Anführungszeichen work Arbeitsverzeichniss, darf Kurzbezeichnung sein
Bemerkung:
Die Definitionen der Kurzbezeichnungen für "centauri-bash-lib" werden in "~/.config" "~" "$CEN_ROOT/default" oder "/etc/default" gesucht.
Beispiele:
centauripaths --list --tabular --verbose # Liste alle Namen/Pfade centauripaths temp # TMPDIR des Benutzers centauripaths --logical automount/alpha4_work # Erstelle logischen Pfad centauripaths --work=/tmp --symlink disk data # Erzeuge Verknüpfungen ... centauripaths --work=/tmp disk --name=d1 data --name=d2
