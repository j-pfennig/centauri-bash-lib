Script - autoload - Create the autoload module _centauri_bash_dyn
Usage:
autoload option... argument...
Create the autoload module _centauri_bash_dyn.
Options:
-d --dryrun show what would be done, make no changes -f --force override prudent default behaviours -h --help show this text -n --no do not prompt for confirmation, assume 'no' -q --quiet report errors only -v --verbose be verbose (use twice for debug) -y --yes do not prompt for confirmation, assume 'yes'
--embed name use name instead of script name --info print full path and version --pager prog pipe output thru pager, use 'none' to disable --trace trace actions() or run() --wait prompt before exit
-B --builtin enable extra bash builtins
Examples:
The recommended usage is:
$ autoload - # include all modules
To generate autoload for selected modules only use:
$ autoload mod... # where <mod> is a 3 letter name
