Topic - Index - Bash Script Library Documentation - centauri-bash-lib
General:
The centauri-bash-lib library was written to simplify the creation of bash scripts that look like standard linux (command line) apps with options, usage help and solid error handling.
Remark:
It is split into modules (bash source files). The main module file is _centauri_bash_lib. The documentation is organized into module, topic and function entries.
