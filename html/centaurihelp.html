Script - centaurihelp - Documentation generator for centauri-bash-lib and centauritools
Usage:
centaurihelp [-A] [-C] [-I] [-M] [-R] [-H folder] [-N] [-U file] ... centaurihelp -T [-H folder] [-N] [-U file] ... centaurihelp -G what [-H folder] ... centaurihelp -D ...
Documentation generator for centauri-bash-lib and centauritools.
This tool can generate documentation in HTML format or can be used at the command line in text mode (like the linux "man" program).
Options:
-d --dryrun show what would be done, make no changes -f --force override prudent default behaviours -h --help show this text -i --include expr include only items matching the expression -n --no do not prompt for confirmation, assume 'no' -q --quiet report errors only -v --verbose be verbose (use twice for debug) -x --exclude expr exclude the items matching the expression -y --yes do not prompt for confirmation, assume 'yes'
--embed name use name instead of script name --info print full path and version --pager prog pipe output thru pager, use 'none' to disable --trace trace actions() or run() --wait prompt before exit
-D --delete delete caches -G --generate what generate HTML documentation -H --html folder create HTML output in folder (implies --all) --file text only: output file names (implies --lines) --lines text only: output line numbers -L --library process library documentation (default) -T --tools process the usage output of tools -U --use file process a single file, "-U ." for 'centaurihelp'
-A --all (re)create missing index/modules/content/reference -C --content produce table-of-content -I --index html only: create 'index.html' -M --modules produce table of modules -R --reference produce function/topic reference
Arguments:
what selection: all | tools | library | dummy folder output folder (- for /dev/stdout, + to prompt) file any tool or library file
Filters:
All extra command line arguments work as output filters. Any number of filters can be used and are cumulative. Tree selectors are provided to restrict output to functions '-' or topics '+' or modules '='.
Examples:
# select generated output centaurihelp --content # output table-of-content centaurihelp --all # also output index/modules/content/reference centaurihelp -i ext # only output _centauri_bash_ext documentation centaurihelp folder att # filter for '*folder*' or '*att*' centaurihelp -M move # in which module is 'move' centaurihelp -C -i _sys # what is in module _centauri_bash_sys centaurihelp - sys # restrict output to functions
# create the documentation as HTML ('doc' is the destination folder) centaurihelp --delete --all --html doc
# create/view the documentation as file (--force enables ANSI escapes) centaurihelp --all --force > doc.txt less -FRX doc.txt
# generate HTML documentation centaurihelp --generate=tools -H mydir # tool docs to custom location centaurihelp --delete --generate=all # clear cache, docs to default
