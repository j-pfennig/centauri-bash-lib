Script - centauricreate - Ein centauri-bash-lib Benutzerskript aus Vorlage erstellen
Aufruf:
centauricreate option... script # Ein Skript erstellen centauricreate --no - # Zeige Optionscache
Ein centauri-bash-lib Benutzerskript aus Vorlage erstellen.
Optionen:
-d --dryrun Zeigen was getan wird, keine Änderungen vornehmen -h --help diesen Text ausgeben -n --no Nicht nach Bestätigung fragen, 'nein' annehmen -q --quiet Nur Fehler melden -v --verbose Viel Ausgabe (zweimal zum Testen) -y --yes Nicht nach Bestätigung fragen, 'ja' annehmen
--embed name Benutze name anstelle des Script Namens --info Vollen Pfad und Version ausgeben --pager prog Programm für seitenweise Ausgabe, 'none' für keines --trace Trace actions() oder run() --wait Nachfrage vor Beenden
-A --author text Setze den Skript Author -D --descr text Kurzbeschreibung des Skripts -L --license text Setze die Skipt Lizenz -N --noconf Konfigurationsdatei nicht ändern -O --option name Setze Option in Vorlage (wiederholbar) -S --shell bash Interpreter überschreiben (shebang) -T --template tmpl Vorlage wählen (Voreinstellung: hello)
Die Werte von --author und --license werden gecached.
Argumente:
bash Ein Pfad, zum Beispiel: /usr/bin/env bash name Optionsname, wird in Grosschreibung konvertiert text Text der kein Semikolon enthält, "-" für Vorgabe tmpl Name einer Vorlage, siehe unten
Vorlagen:
action Vorlage mit Kommandozeilen-Verben (action Syntax) cgiforms Demonstrates CGI programming using the XML module daemon Example daemon or service files Vorlage für ein Dateiprogramm (keine 'action' Syntax) hello A Hello World Example run Template for simple scripts (no action syntax) service Example oneshot service simple Using _centauri_bash_lib without main() module a custom library module
Beispiele:
# Von Vorlage 'centauriHello' Skript ohne 'main()' erstellen centauricreate --shell='/usr/bin/env bash' myfolder/simple
# Erstelle Skript mit 'action' Syntax centauricreate --template=action mytool
# Erstelle Skript ohne Demo Code centauricreate --template=action --option=minimal mytool
