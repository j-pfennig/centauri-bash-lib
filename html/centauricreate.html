Script - centauricreate - Create a centauri-bash-lib user script from a template
Usage:
centauricreate option... script # create a script centauricreate --no - # show cached options
Create a centauri-bash-lib user script from a template.
Options:
-d --dryrun show what would be done, make no changes -h --help show this text -n --no do not prompt for confirmation, assume 'no' -q --quiet report errors only -s --silent generate less output or warnings -v --verbose be verbose (use twice for debug) -y --yes do not prompt for confirmation, assume 'yes'
-A --author text set script author -D --descr text short description of script -L --license text set script license -N --noconf don't modify config file -O --option name select template option (repeatable) -S --shell bash override interpreter path (shebang) -T --template tmpl choose template (default: hello)
The values of --author and --license are cached.
Arguments:
bash any path, for example: /usr/bin/env bash name option name, gets converted to uppercase text any text not containing ";" or "-" to reset tmpl name of a template, see below
Templates:
action Template using command line verbs (action syntax) cgiforms Demonstrates CGI programming using the XML module daemon Example daemon or service files Template for a file processor (no action syntax) hello A Hello World Example run Template for simple scripts (no action syntax) service Example oneshot service simple Using _centauri_bash_lib without main() module a custom library module
Examples:
# create script without 'main()' from centauriHello template centauricreate --shell='/usr/bin/env bash' myfolder/simple
# create script with 'action' syntax centauricreate --template=action mytool
# create script without demo code centauricreate --template=action --option=minimal mytool
