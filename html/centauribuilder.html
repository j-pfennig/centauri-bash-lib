Script - centauribuilder - Tool to build packaged or portable applications
Usage:
centauribuilder option... [action argument... [--]]...
Tool to build packaged or portable applications.
Options:
-d --dryrun show what would be done, make no changes -f --force override prudent default behaviours -h --help show this text -n --no do not prompt for confirmation, assume 'no' -q --quiet report errors only -v --verbose be verbose (use twice for debug) -y --yes do not prompt for confirmation, assume 'yes' -z --zombie next action(s) even after error
--embed name use name instead of script name --info print full path and version --pager prog pipe output thru pager, use 'none' to disable --trace trace actions() or run() --wait prompt before exit
-D --descr text add/create only: short description of the script -F --flavour flav add/create only: build flavour to use -T --template tmpl add/create only: template to use -X --extra file add/create only: extra file, repeatable
Actions:
••• working with projects ••• create work [name] # create new workspace [and project] add work name # add new project to workspace folder add work path # import script to workspace folder remove work name # remove tool from project folder package [work name] # build package file runtime [work name] # create a runtime file ... join [work name] # join package and runtime list [work [name]] # list runtime content [build] [work name] # build tool(s), shorthand for actions: # package ... -- runtime ... -- join ... ••• non-project use ••• package pack tool modu... # create self-extracting script runtime runt file... # create self-extracting runtime data join outp pack runt # join package and runtime list outp|runt # list runtime content
Arguments:
file... extra file(s) to be added to runtime flav tool flavour: auto package portable archive work workspace folder for centauribuilder projects modu... list of (static) modules, passed to uglify name project name (no '.' characters please) outp file for joined script and runtime data pack packaged script file, see uglify tool path a centauri-bash-lib script path (with /) runt file with runtime self-extract data tool a centauri-bash-lib script
General:
This tool is a high-level build engine for stand-alone scripts. It runs tools like centauricreate, uglify, l10n-tool and tar. It maintains a work folder for imported or local projects. The projects are described by configuration files. These *.conf files are created for new projects and contain comments describing the configuration.
Examples:
# create a project workspace centauribuilder create standalone
# add a newly created script centauribuilder -D "a new script" add standalone ./new-tool -- package
# import an existing script, build and run it centauribuilder add standalone ./my-tool -- build standalone/my-tool.package
