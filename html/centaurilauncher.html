Script - centaurilauncher - Launcher for GUI or Console Applications
Usage:
centaurilauncher option... [action argument... [--]]...
Launcher for GUI or Console Applications.
Options:
-d --dryrun show what would be done, make no changes -h --help show this text -n --no do not prompt for confirmation, assume 'no' -q --quiet report errors only -v --verbose be verbose (use twice for debug) -y --yes do not prompt for confirmation, assume 'yes' -z --zombie next action(s) even after error
--debug mask debug mode (bit 1:=modules 2:=calls 4:=trace)
-M --mode mode start mode (run in background) -D --desktop desk override desktop configuration to use
Actions:
browser arg... open a web browser console arg... command line terminal editor arg... open a text editor files arg... open a file manager httpd arg... launch a private web server kiosk arg... web browser in kiosk mode library arg... interactive centauri-bash-lib console viewer arg... html viewer (qml or browser) webapp arg... run a centauri-bash-lib web application
list info show detailed information about actions
Arguments:
desk a desktop name like: tty kde lxqt gnome info one out of: all session executable mode 0:wait (default) 1:exec 2:background 3:disown
