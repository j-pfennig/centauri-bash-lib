Script - centaurilauncher - Starter für GUI- und Konsolen-Anwendungen
Centaurihelp:
configfile: Datei lesen: /etc/default/centauri-bash-lib configfile: Datei lesen: /etc/default/centauri-bash-lib configfile: Datei lesen: /etc/default/centauri-bash-lib serialize: write to: /home/local/.cache/centauri-bash-lib.dat Aufruf: centaurilauncher option... [action argument... [--]]...
Starter für GUI- und Konsolen-Anwendungen.
Optionen:
-d --dryrun Zeigen was getan wird, keine Änderungen vornehmen -h --help diesen Text ausgeben -n --no Nicht nach Bestätigung fragen, 'nein' annehmen -q --quiet Nur Fehler melden -v --verbose Viel Ausgabe (zweimal zum Testen) -y --yes Nicht nach Bestätigung fragen, 'ja' annehmen -z --zombie Nächste action(s) auch nach Fehler
--embed name Benutze name anstelle des Script Namens --info Vollen Pfad und Version ausgeben --pager prog Programm für seitenweise Ausgabe, 'none' für keines --trace Trace actions() oder run() --wait Nachfrage vor Beenden
--debug mask debug mode (bit 1:=modules 2:=calls 4:=trace)
-M --mode mode Start Modus (in Hintergrund laufen) -D --desktop desk Verwendendete Desktopeinstellung ändern
Aktionen:
browser arg... open a web browser console arg... command line terminal editor arg... open a text editor files arg... open a file manager httpd arg... launch a private web server kiosk arg... web browser in kiosk mode library arg... interactive centauri-bash-lib console webapp arg... run a centauri-bash-lib web application
list info Detailinformationen zu Aktionen anzeigen
Argumente:
desk Ein Desktop Name wie: tty kde lxqt gnome info einer von: all session executable mode 0:warten (Voreinst.) 1:exec 2:Hintergrund 3:disown
